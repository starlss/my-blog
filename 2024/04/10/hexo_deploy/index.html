部署 Hexo 博客到 GitHub Pages

在本教程中，我们将学习如何将 Hexo 博客部署到 GitHub Pages，以便让公众可以访问你的博客。

准备工作

在开始之前，请确保你已经安装了以下软件和工具：

  • Git
  • Node.js
  • Hexo

创建 Hexo 博客

如果你还没有 Hexo 博客，可以通过以下命令创建一个新的 Hexo 博客：

hexo init my-blog
cd my-blog
npm install
